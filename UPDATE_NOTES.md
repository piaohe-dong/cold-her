# 游戏更新说明

## 版本 1.1 - UI优化版

### 🎯 主要改进

#### 1. AI玩家显示优化
**问题**: 之前AI玩家的卡牌全部展示,信息过多
**解决方案**:
- AI玩家仅显示:名称 + 手牌数量 + 质疑卡牌数量
- 使用简洁的机器人图标 Avatar 展示
- 不再展示AI的具体手牌内容
- 保持游戏公平性和悬念

**效果**:
```
┌─────────────────────────┐
│ AI玩家1 [AI]            │
│ 🎴 5  |  🔍 2           │
│                         │
│   🤖  手牌: 5 张        │
│       质疑: 2 张         │
└─────────────────────────┘
```

#### 2. 技能目标选择流程优化
**问题**: 之前选择目标玩家时没有清晰的交互流程
**解决方案**:
- 实现两步选择流程:
  1. **第一步**: 弹出目标选择器,选择目标玩家
  2. **第二步**(如需要): 显示该玩家的手牌序号列表,选择具体卡牌
- 支持的技能类型:
  - 查看1张手牌(学生会长)
  - 查看所有手牌(风纪委员)
  - 交换1张牌(图书委员)
  - 禁止技能(保健委员)
  - 查看身份(新闻部)
  - 强制调和(体育委员)
  - 给予卡牌(共犯)
  - 质疑玩家

**目标选择器特性**:
- 清晰显示所有可选玩家
- 显示每个玩家的手牌数和质疑数
- 高亮已选中的玩家
- 支持手牌序号选择(当技能需要时)
- 确认/取消按钮

#### 3. 调和区布局重构
**问题**: 调和区在左侧,不够醒目
**解决方案**:
- 调和区现在**固定在屏幕正中央**
- 使用固定定位(position: fixed),始终可见
- 不遮挡玩家区域,玩家可以正常交互
- 视觉层级更清晰

**调和区显示内容**:
```
┌──────────────────────┐
│  🧪 调和区            │
│                      │
│  卡牌数量: 3 张       │
│  目标调和值: 10       │
│                      │
│  [进度条: ████░░░]   │
│  7 / 10              │
│                      │
│  [查看调和区] 按钮    │
└──────────────────────┘
```

#### 4. 调和区查看功能增强
**问题**: 无法追踪调和区的变化
**解决方案**:
- 实现**缓存机制**:
  - 点击"查看调和区"按钮,显示所有卡牌5秒钟
  - 自动记录当前调和值
  - 5秒后自动隐藏卡牌
  
- **变化检测**:
  - 如果调和区有新增卡牌,显示 `上次值 + ?`
  - 例如: 上次查看是7,新增卡牌后显示 `7 + ?`
  - 再次查看后更新缓存值

**显示状态**:
- 未查看过: `? / 10`
- 已查看(无变化): `7 / 10`
- 已查看(有变化): `7 + ? / 10`
- 正在查看: 显示所有卡牌正面,5秒后自动隐藏

### 📋 使用指南

#### 使用技能
1. 点击"发动技能"按钮
2. 点击要使用的卡牌
3. 弹出目标选择器,选择目标玩家
4. 如果技能需要,继续选择该玩家的具体手牌
5. 点击确认执行

#### 查看调和区
1. 调和区有卡牌时,显示"查看调和区"按钮
2. 点击查看,显示所有卡牌正面
3. 5秒后自动隐藏
4. 如果期间有变化,显示 `上次值 + ?`

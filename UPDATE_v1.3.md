# 游戏更新说明 - 版本 1.3

## 🎯 本次更新内容

### 1. 调和区域优化

#### 改进前
- 调和区占据整个顶部区域,空间利用率低
- 显示所有卡牌详情,信息过载
- 进度条过于占用空间

#### 改进后
**精简面板设计**:
```
┌──────────────────────┐
│  🧪 调和区            │
│                      │
│  卡牌: 3 张          │
│  目标: 10            │
│  当前: 7 + ?         │
│                      │
│  [████████░░]        │
│                      │
│  [查看卡牌]          │
└──────────────────────┘
```

**核心特性**:
- ✅ 紧凑布局,只显示关键信息
- ✅ 三行信息: 卡牌数量、目标值、当前值
- ✅ 迷你进度条,直观显示进度
- ✅ 查看卡牌按钮,点击后显示5秒
- ✅ 智能显示:
  - 查看中: 显示实际数值 (如 `7`)
  - 有变化: 显示上次值+问号 (如 `7 + ?`)
  - 未查看: 显示问号 (如 `?`)
  - 达标时数值变绿色

### 3. 布局重构

#### 新的页面结构

**之前**:
```
┌─────────────────────────────────┐
│  顶部信息栏                      │
├─────────────────────────────────┤
│  调和区 (全宽)                   │
├─────────────────────────────────┤
│  玩家区域 (全宽)                 │
├─────────────────────────────────┤
│  手牌区 (底部)                   │
├─────────────────────────────────┤
│  行动面板 (底部)                 │
└─────────────────────────────────┘
```

**现在**:
```
┌─────────────────────────────────────────────────────┐
│  顶部信息栏                                          │
├──────────────────────────┬──────────────────────────┤
│  玩家区域 (自适应宽度)    │  右侧面板 (固定300px)     │
│                          ├──────────────────────────┤
│  [玩家1] [玩家2] [玩家3] │  🧪 调和区 (精简)         │
│  [玩家4] [玩家5] [AI]    ├──────────────────────────┤
│                          │  🏆 胜利条件              │
│                          │                          │
├──────────────────────────┴──────────────────────────┤
│  我的手牌 (居中展示)                                 │
├─────────────────────────────────────────────────────┤
│  行动面板                                            │
└─────────────────────────────────────────────────────┘
```

**布局优势**:
1. ✅ **空间利用更高效**: 调和区从全宽缩小到300px
2. ✅ **信息层级清晰**: 玩家区域占据主要空间
3. ✅ **参考信息常驻**: 胜利条件随时可见,无需翻阅规则
4. ✅ **响应式设计**: 玩家区域自适应宽度,支持不同屏幕尺寸

## 📊 技术细节

### 新增数据结构

```typescript
// 胜利条件数据
const winConditions = [
  { faction: '宇宙人', desc: '被监禁即可获胜', priority: 1 },
  { faction: '感染者', desc: '调和失败即可获胜', priority: 2 },
  { faction: '犯人与共犯', desc: '犯人未被监禁', priority: 3 },
  { faction: '学生阵营', desc: '调和成功且犯人被监禁', priority: 4 },
  { faction: '归宅部', desc: '自己未被监禁且调和失败', priority: 5 }
```

### CSS 类名说明

- `.right-panel`: 右侧固定宽度面板容器
- `.harmony-panel-compact`: 精简版调和区
- `.win-conditions-panel`: 胜利条件面板
- `.condition-item.priority-{1-5}`: 不同优先级的样式类

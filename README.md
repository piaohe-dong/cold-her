# 在冰冷的她醒来之前 - 桌游电子版

基于Vue 3 + TypeScript开发的《在冰冷的她醒来之前》桌游电子版,支持本地多人和AI对手模式。

## 🎮 游戏特色

- **本地多人**: 支持3-6人在同一设备上轮流游戏
- **AI对手**: 可配置1-5个AI对手,基于策略自动决策
- **完整规则**: 实现原桌游的所有规则和卡牌效果,包含完整的技能系统
- **已使用卡牌区域**: 发动技能后的卡牌正面展示在玩家面前,可随时查看
- **简约UI**: 清爽现代的界面设计,支持响应式布局
- **实时反馈**: 游戏日志实时记录所有行动

## 📦 技术栈

- Vue 3 (Composition API)
- TypeScript
- Vite
- Vue Router 4
- 纯CSS (无UI框架)

## 🎯 游戏玩法

### 游戏设置

1. 选择玩家人数(3-6人)
2. 选择AI对手数量(至少保留1个真实玩家)
3. 点击"开始游戏"

### 回合行动

每位玩家在自己的回合必须选择以下三种行动之一:

1. **⚡ 发动技能**: 正面打出卡牌,卡牌放置在玩家面前的"已使用卡牌"区域,执行卡牌上的技能效果
2. **🧪 调和尸体**: 背面朝上打出卡牌,放入调和区增加强度值
3. **🔍 质疑玩家**: 背面朝上打出卡牌,放置到其他玩家面前的"质疑区"表达怀疑

### 身份锁定

当玩家手牌只剩1张时:
- 该牌自动成为身份牌
- 玩家暂时退出游戏
- 手牌被锁定,不受其他技能影响

### 游戏结束

当所有玩家都只剩1张手牌时,游戏结束,进入三阶段结算:

1. **调和判定**: 计算调和区卡牌强度总和,达到目标值则调和成功
2. **监禁判定**: 质疑区强度总和最高的玩家被监禁
3. **身份结算**: 按优先级判定胜利阵营

### 胜利条件(优先级从高到低)

1. **外星人**: 被监禁即可获胜
2. **感染者**: 调和失败即可获胜
3. **犯人与共犯**: 犯人未被监禁
4. **学生阵营**: 调和成功且犯人被监禁
5. **归宅部**: 无任何人胜利即可获胜

## 🃏 卡牌说明

游戏包含25张卡牌,分为6个阵营:

### 学生阵营(调和成功且犯人被监禁)
- 学生会长(1张): 作为起始玩家(无技能)
- 班长(2张): 和1名玩家互换1张手牌
- 优等生(2张): 可以知道犯人在哪(查看身份牌)
- 风纪委员(2张): 可以查看某人的全部手牌
- 保健委员(2张): 可以夺走1张已使用的牌
- 图书委员(3张): 可以查看调和位置的牌
- 大小姐(3张): 夺走他人1张手牌并返还1张
- 新闻部(3张): 所有人将1张手牌传给左边玩家

### 犯人阵营(犯人未被监禁)
- 犯人(1张): 不能使用(无技能)
- 共犯(1张): 移动1张质疑位置的牌

### 特殊阵营
- 外星人(1张): 持有期间,可装作犯人(被监禁即可获胜)
- 感染者(1张): 下回合可夺走1张调和位置的牌(调和失败即可获胜)
- 归宅部(3张): 1张手牌交换1张调和位置的牌(无任何人胜利即可获胜)

## 🔧 开发说明

### 添加新卡牌

在 `src/data/cards.ts` 中的 `ALL_CARDS` 数组添加卡牌定义:

```typescript
{
  name: '卡牌名称',
  strength: 强度值,
  skill: '技能说明',
  skillType: SkillType.技能类型,
  winCondition: '胜利条件',
  faction: Faction.阵营,
  count: 数量
}
```

### 修改AI策略

编辑 `src/logic/ai.ts` 中的 `evaluateStrategy` 和决策函数来调整AI行为。

### 调整调和目标

在 `src/types/game.ts` 中修改 `HARMONY_TARGETS` 常量。

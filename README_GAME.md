# 在冰冷的她醒来之前 - 桌游电子版

基于Vue 3 + TypeScript开发的《在冰冷的她醒来之前》桌游电子版,支持本地多人和AI对手模式。

## 🎮 游戏特色

- **本地多人**: 支持3-6人在同一设备上轮流游戏
- **AI对手**: 可配置1-5个AI对手,基于策略自动决策
- **完整规则**: 实现原桌游的所有规则和卡牌效果
- **简约UI**: 清爽现代的界面设计,支持响应式布局
- **实时反馈**: 游戏日志实时记录所有行动

## 📦 技术栈

- Vue 3 (Composition API)
- TypeScript
- Vite
- Vue Router 4
- 纯CSS (无UI框架)

## 🎯 游戏玩法

### 游戏设置

1. 选择玩家人数(3-6人)
2. 选择AI对手数量(至少保留1个真实玩家)
3. 点击"开始游戏"

### 回合行动

每位玩家在自己的回合必须选择以下三种行动之一:

1. **⚡ 发动技能**: 正面打出卡牌,执行卡牌上的技能效果
2. **🧪 调和尸体**: 背面朝上打出卡牌,放入调和区增加强度值
3. **🔍 质疑玩家**: 背面朝上打出卡牌,放置到其他玩家面前表达怀疑

### 身份锁定

当玩家手牌只剩1张时:
- 该牌自动成为身份牌
- 玩家暂时退出游戏
- 手牌被锁定,不受其他技能影响

### 游戏结束

当所有玩家都只剩1张手牌时,游戏结束,进入三阶段结算:

1. **调和判定**: 计算调和区卡牌强度总和,达到目标值则调和成功
2. **监禁判定**: 质疑区强度总和最高的玩家被监禁
3. **身份结算**: 按优先级判定胜利阵营

### 胜利条件(优先级从高到低)

1. **宇宙人**: 被监禁即可获胜
2. **感染者**: 调和失败即可获胜
3. **犯人与共犯**: 犯人未被监禁
4. **学生阵营**: 调和成功且犯人被监禁
5. **归宅部**: 自己未被监禁且调和失败

## 🃏 卡牌说明

游戏包含25张卡牌,分为6个阵营:

### 学生阵营(调和成功且犯人被监禁)
- 学生会长(1张): 查看任意1名玩家的1张手牌
- 风纪委员(2张): 查看任意1名玩家的所有手牌
- 图书委员(3张): 从任意玩家手中抽1张牌,然后给其1张牌
- 班长(2张): 交换任意两名玩家的所有手牌
- 保健委员(2张): 选择1名玩家,其本回合不能发动技能
- 文艺委员(2张): 查看调和区最上方1张卡牌
- 体育委员(2张): 指定1名玩家,其必须将1张手牌作为调和牌打出
- 新闻部(2张): 查看任意1名玩家的最终身份牌
- 优等生(2张): 将自己的1张手牌与牌库顶牌交换
- 普通学生(3张): 无特殊技能

### 犯人阵营(犯人未被监禁)
- 犯人(1张): 无特殊技能
- 共犯(1张): 可将1张手牌交给任意1名玩家

### 特殊阵营
- 宇宙人(1张): 被监禁即可获胜
- 感染者(1张): 调和失败即可获胜
- 归宅部(1张): 自己未被监禁且调和失败即可获胜

## 🔧 开发说明

### 添加新卡牌

在 `src/data/cards.ts` 中的 `ALL_CARDS` 数组添加卡牌定义:

```typescript
{
  name: '卡牌名称',
  strength: 强度值,
  skill: '技能说明',
  skillType: SkillType.技能类型,
  winCondition: '胜利条件',
  faction: Faction.阵营,
  count: 数量
}
```

### 修改AI策略

编辑 `src/logic/ai.ts` 中的 `evaluateStrategy` 和决策函数来调整AI行为。

### 调整调和目标

在 `src/types/game.ts` 中修改 `HARMONY_TARGETS` 常量。
